Desafio Futuro

## Como rodar o projeto:

### Instalar as dependencias
`````js
yarn install
`````

### Rodar o app
`````js
yarn android 

or

yarn ios
`````

### Rodar apk in emulator

`````js
expo build:android -t apk
`````
